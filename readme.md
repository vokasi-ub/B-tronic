## How To Install B-tronic

- git clone https://github.com/vokasi-ub/B-tronic.git
- on cmd project type
- 'composer install'
- 'php artisan key : generate'
- 'php artisan migrate'
- 'php artisan db:seed --class=UsersSeeder'
- 'php artisan laravolt : indonesia : seed'
- then 'php artisan serve'
